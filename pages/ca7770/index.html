JS与Node异同

# JavaScript

  • ECMAScript（语言基础，如：语法、数据类型结构以及一些内置对象）
  • BOM（一些操作浏览器的方法）
  • DOM（一些操作页面元素的方法）

# Nodejs

  • ECMAScript (语言基础，如：语法、数据类型结构以及一些内置对象)
  • os (操作系统)
  • file (文件系统)
  • net (网络系统)
  • database (数据库)

# NodeJS 特点

事件驱动、异步处理、非I/O阻塞等。
