# polyfill

概念：用于实现浏览器并不支持原生 API 的代码。

ployfill 和 shim 的区别：

  • 一个 shim 是一个库，它将一个新的 API 引入到一个旧的环境中，而且仅靠旧环境中已有的手段实现。
  • 一个 polyfill 就是一个用在浏览器 API 上的 shim。
